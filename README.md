# CodeAlpha_ImageGallery
Student Information:
Bandaru Pavani is a 2nd Year B.Tech CSE student interested in frontend development and web technologies.

Project Overview:
The Creative Image Gallery is a responsive web application developed using HTML, CSS, and JavaScript with image filtering, hover effects, and lightbox functionality.

Features:
- Responsive Design
- Image Filter Buttons
- Hover Effects
- Lightbox Preview
- Next/Previous Navigation

Technologies Used:
- HTML5
- CSS3
- JavaScript

Conclusion:
This project improves frontend development skills and provides practical experience in creating interactive web applications.
